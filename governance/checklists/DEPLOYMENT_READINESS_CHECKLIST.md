# Deployment Readiness Checklist

## Governance
- [ ] Use case approved and risk-classified
- [ ] Owner and accountability assigned
- [ ] Policies acknowledged by operators

## Security
- [ ] Access controls configured and tested
- [ ] Secrets managed securely
- [ ] Logging and alerting enabled
- [ ] Backup/recovery tested

## AI/Agent controls
- [ ] Autonomy tier assigned
- [ ] Approval gates configured
- [ ] Budget/concurrency limits set
- [ ] Safe tool and integration policies applied

## Validation
- [ ] Functional testing passed
- [ ] Security testing passed
- [ ] Rollback procedure validated
