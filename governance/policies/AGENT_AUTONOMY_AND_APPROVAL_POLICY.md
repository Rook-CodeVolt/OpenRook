# Agent Autonomy and Approval Policy (Template)

## Risk tiers
- Tier 0: Observe only
- Tier 1: Suggest only
- Tier 2: Auto low-risk
- Tier 3: Approval required before action

## Approval requirements
- Any action affecting production availability, security posture, legal/compliance, or external communications requires approval unless explicitly pre-authorized.

## Runtime constraints
- Per-agent budgets
- Concurrency caps
- Mandatory audit logs
- Kill switch support
